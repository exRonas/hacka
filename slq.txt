<div class="theme_header"> <h1 class="theme_title text-center mb-4">HTML для всех: Первые шаги в веб-программировании</h1> </div> <div class="theme_main"> <section> <h2 class="theory_title">Что такое HTML?</h2> <p class="theory_text"> HTML (HyperText Markup Language) - это язык разметки, который используется для определения структуры и содержания веб-страницы. </p> </section> <section> <h2 class="theory_title">Первый HTML документ</h2> <p class="theory_text"> Давайте начнем с создания нашего первого HTML документа. HTML документ состоит из двух основных частей: тега &lt;head&gt; и тега &lt;body&gt;. </p> <button img="code_2.png" class="theory_button code_show">Показать код</button> <p class="theory_text">Разбор кода:</p> <ul> <li> <code>&lt;!DOCTYPE html&gt;</code>: Эта строка указывает браузеру на то, что мы используем последнюю версию HTML. </li> <li> <code>&lt;html&gt;</code>: Этот тег обозначает начало HTML документа. </li> <li> <code>&lt;head&gt;</code>: В этом блоке содержится метаданные документа. </li> <li> <code>&lt;title&gt;</code>: Внутри этого тега мы указываем заголовок страницы, который отображается во вкладке браузера. </li> <li> <code>&lt;body&gt;</code>: Здесь размещается основной контент страницы. </li> <li> <code>&lt;h1&gt;</code>: Это тег для создания заголовка первого уровня. </li> <li><code>&lt;p&gt;</code>: Это тег для создания абзаца текста.</li> </ul> </section> <section> <h2 class="theory_title">Давайте начнем!</h2> <p class="theory_text"> Теперь, когда мы знаем основы HTML, давайте попробуем создать свою первую веб-страницу. Используйте пример кода выше, чтобы создать свой собственный HTML документ. Помните, что HTML - это весело! Вы можете экспериментировать с разными тегами и структурами, чтобы создать уникальные и интересные веб-страницы. </p> </section> </div>

<div class="theme_header"> <h1 class="theme_title text-center mb-4">Изображения и мультимедиа в HTML</h1> </div> <div class="theme_main"> <section> <h2 class="theory_title">Как вставить изображение на веб-страницу</h2> <p class="theory_text"> <p>Чтобы добавить изображение на веб-страницу, нужно использовать тег <code>&lt;img&gt;</code>. Этот тег имеет атрибут <code>src</code>, который указывает путь к изображению.</p> <p>Вот пример кода, который вставляет изображение с именем "myimage.jpg":</p> <div class="code"> &lt;img src="myimage.jpg" alt="Мое изображение"&gt; </div> <p>В атрибуте <code>src</code> указывается имя файла изображения, а в атрибуте <code>alt</code> - альтернативный текст, который отображается, если изображение не может быть загружено.</p> </p> </section> <section> <h2 class="theory_title">Как вставить аудио на веб-страницу</h2> <p class="theory_text"> <p>Для вставки аудиофайла используется тег <code>&lt;audio&gt;</code>. Этот тег имеет атрибут <code>src</code>, который указывает путь к аудиофайлу.</p> <p>Вот пример кода, который вставляет аудиофайл с именем "mysong.mp3":</p> <div class="code"> &lt;audio controls&gt; &lt;source src="mysong.mp3" type="audio/mpeg"&gt; Ваш браузер не поддерживает аудиофайлы. &lt;/audio&gt; </div> <p>Внутри тега <code>&lt;audio&gt;</code> можно использовать один или несколько тегов <code>&lt;source&gt;</code> для указания разных форматов аудиофайла для разных браузеров.</p> </p> </section> <section> <h2 class="theory_title">Как вставить видео на веб-страницу</h2> <p class="theory_text"> <p>Для вставки видео используется тег <code>&lt;video&gt;</code>. Как и у тега <code>&lt;audio&gt;</code>, у него также есть атрибут <code>src</code>.</p> <p>Вот пример кода, который вставляет видео с именем "myvideo.mp4":</p> <div class="code"> &lt;video controls width="400"&gt; &lt;source src="myvideo.mp4" type="video/mp4"&gt; Ваш браузер не поддерживает видеофайлы. &lt;/video&gt; </div> <p>Как и с аудиофайлами, можно использовать несколько тегов <code>&lt;source&gt;</code> для разных форматов видео.</p> </p> </section> </div>

<div class="theme_header"> <h1 class="theme_title text-center mb-4">Работа с таблицами в HTML</h1> </div> <div class="theme_main"> <section> <h2 class="theory_title">Как создать таблицу в HTML</h2> <p> Чтобы создать таблицу на веб-странице, нужно использовать теги <code>&lt;table&gt;</code>, <code>&lt;tr&gt;</code> и <code>&lt;td&gt;</code>. </p> <p> Тег <code>&lt;table&gt;</code> определяет саму таблицу, тег <code>&lt;tr&gt;</code> - строки в таблице, а тег <code>&lt;td&gt;</code> - ячейки в строках. </p> <p>Вот пример кода, который создает простую таблицу:</p> <div class="code"> &lt;table border="1"&gt;<br /> &emsp;&lt;tr&gt;<br /> &emsp;&emsp;&lt;td&gt;Ячейка 1&lt;/td&gt;<br /> &emsp;&emsp;&lt;td&gt;Ячейка 2&lt;/td&gt;<br /> &emsp;&lt;/tr&gt;<br /> &emsp;&lt;tr&gt;<br /> &emsp;&emsp;&lt;td&gt;Ячейка 3&lt;/td&gt;<br /> &emsp;&emsp;&lt;td&gt;Ячейка 4&lt;/td&gt;<br /> &emsp;&lt;/tr&gt;<br /> &lt;/table&gt; </div> <p> В атрибуте <code>border</code> можно указать толщину границ таблицы (1 - самая тонкая, 2 - более толстая и т. д.). </p> </section> <section> <h2 class="theory_title">Как объединить ячейки в таблице</h2> <p class="theory_text"> <p> Чтобы объединить ячейки в таблице, нужно использовать атрибуты <code>rowspan</code> и <code>colspan</code>. </p> <p> Атрибут <code>rowspan</code> позволяет объединить ячейки в вертикальном направлении, а <code>colspan</code> - в горизонтальном. </p> <p>Вот пример кода, который объединяет ячейки в таблице:</p> <div class="code"> &lt;table border="1"&gt;<br /> &emsp;&lt;tr&gt;<br /> &emsp;&emsp;&lt;td colspan="2"&gt;Объединенная ячейка&lt;/td&gt;<br /> &emsp;&lt;/tr&gt;<br /> &emsp;&lt;tr&gt;<br /> &emsp;&emsp;&lt;td&gt;Ячейка 2.1&lt;/td&gt;<br /> &emsp;&emsp;&lt;td&gt;Ячейка 2.2&lt;/td&gt;<br /> &emsp;&lt;/tr&gt;<br /> &lt;/table&gt; </div> <p> Этот код создает таблицу, в которой первая ячейка в первой строке объединена с двумя следующими ячейками. </p> </p> </section> </div>

<div class="theme_header"> <h1 class="theme_title text-center mb-4">Формы и ввод пользователя в HTML</h1> </div> <div class="theme_main"> <section> <h2 class="theory_title">Как создать форму в HTML</h2> <p class="theory_text"> <p>Чтобы создать форму на веб-странице, нужно использовать тег <code>&lt;form&gt;</code>. Внутри формы могут быть различные элементы, такие как текстовые поля, кнопки, флажки и т. д.</p> <p>Вот пример кода, который создает простую форму с текстовым полем и кнопкой:</p> <div class="code"> &lt;form&gt;<br> &emsp;&lt;label for="name"&gt;Имя:&lt;/label&gt;<br> &emsp;&lt;input type="text" id="name" name="name"&gt;<br> &emsp;&lt;input type="submit" value="Отправить"&gt;<br> &lt;/form&gt; </div> <p>Этот код создает форму с текстовым полем для ввода имени и кнопкой "Отправить".</p> </p> </section> <section> <h2 class="theory_title">Как объединить ячейки в таблице</h2> <p class="theory_text"> <p>Для создания списка выбора используется тег <code>&lt;select&gt;</code>, а для определения элементов списка - тег <code>&lt;option&gt;</code>.</p> <p>Вот пример кода, который создает выпадающий список с несколькими опциями:</p> <div class="code"> &lt;select id="color" name="color"&gt;<br> &emsp;&lt;option value="red"&gt;Красный&lt;/option&gt;<br> &emsp;&lt;option value="green"&gt;Зеленый&lt;/option&gt;<br> &emsp;&lt;option value="blue"&gt;Синий&lt;/option&gt;<br> &lt;/select&gt; </div> <p>Этот код создает выпадающий список с тремя опциями: "Красный", "Зеленый" и "Синий".</p> </p> </section> <section> <h2 class="theory_title">Как создать флажок в HTML</h2> <p class="theory_text"> <p>Для создания флажка (checkbox) используется тег <code>&lt;input&gt;</code> с атрибутом <code>type="checkbox"</code>.</p> <p>Вот пример кода, который создает флажок:</p> <div class="code"> &lt;input type="checkbox" id="subscribe" name="subscribe"&gt;<br> &lt;label for="subscribe"&gt;Подписаться на рассылку&lt;/label&gt; </div> <p>Этот код создает флажок "Подписаться на рассылку".</p> </p> </section> </div>
